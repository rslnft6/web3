الخطوات:
  1. اختر ملف لرفعه (مثال: README.md أو أي ملف آخر)

  3. بعد الدفع، يتم رفع الملف إلى IPFS وWeb3Storage
  4. يظهر رابط الملف ويمكنك مشاركته أو نشره مباشرة
